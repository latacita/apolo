Descargas

Apolo ha sido desarrollado de manera que fuera multiplataforma. Requiere tener instalada la maquina virtual de Java, version 1.6.0 o superior. Para facilitar la instalacion en los sistemas. Se han puesto a disposicion de los usuarios paquetes instaladores tanto para sistemas operativos micorsoft Windows como Linux basados en Debian.

Descargas
18 de Julio 2011

- Instalador para sistemas Microsoft Windows

- Instalador para sistemas Linux basados en Debian

- Resto de sistemas (Paquete con ejecutable "Jar")

- Codigo Fuente

Requisitos Minimos

  • Sistema compatible con Java
  • Java version: 1.6.0
  • Resolucion de 1024x768
  • 512MB de Memoria
