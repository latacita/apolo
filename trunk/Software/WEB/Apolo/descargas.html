Descargas

Apolo ha sído desarrollado de manera que fuera multiplataforma. Requiere tener instalada la máquina virtual de Java, versión 1.6.0 o superior. Para facilitar la instalación en los sistemas, se han puesto a disposición de los usuarios paquetes instaladores, tanto para sistemas operativos Micorsoft Windows, como Linux basados en Debian.

Descargas
18 de Julio 2011

- Instalador para sistemas Microsoft Windows

- Instalador para sistemas Linux basados en Debian

- Resto de sistemas (Paquete con ejecutable "Jar")

- Código Fuente

Requisitos Mínimos

  • - Sistema compatible con Java
  • - Java versión: 1.6.0
  • - Resolución mayor que 1024x768
  • - 512MB de Memoria
