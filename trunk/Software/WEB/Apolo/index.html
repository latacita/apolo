Apolo

Apolo es un clasificador de fotografias digitales, basado en el funcionamiento de un clasificador de diapositivas analogicas.

Apolo
18 de Julio 2011

Apolo fue desarrollado como Proyecto Fin de Carrera de Angel Tezanos Ibañez, el cual fue supervisado por Pablo Sanchez Barreiro. El software se diseñó con el fin de disponer de una herramienta capaz de organizar y clasificar fotografías como si se trataran de diapositivas. Para ello se diseñó una interfaz gráfica que recordase a un clasificador de diapositivas clásico.

Su funcionamiento es sencillo, y consiste, al igual que los viejos calsificadores, en arrastrar y colocar las fotografias que se deseen, a baldas virtuales. Una vez que una balda haya sido ordenada y clasificada se pueden empaquetar virtualmente. Cuando tengamos varios paquetes, podemos ordenar el orden de estos, y finalmente exportar las fotografias a un nuevo directorio conservando el orden y clasificacion que habiamos realizado.
