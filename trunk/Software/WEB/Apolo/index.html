Apolo

Apolo es un clasificador de fotografías digitáles, basado en el funcionamiento de un clasificador de diapositívas analógicas.

Apolo
18 de Julio 2011

Apolo fue desarrollado como Proyecto Fin de Carrera de Angel Tezanos Ibáñez, el cual fue supervisado por Pablo Sánchez Barreiro. El software se diseñó con el fin de disponer de una herramienta capaz de organizar y clasificar fotografías como si se trataran de diapositívas. Para ello se diseñó una interfaz gráfica que recordase a un clasificador de diapositívas clásico.

Su funcionamiento es sencillo, y consiste, al igual que los viejos clasificadores, en arrastrar y colocar las fotografías que se deseen, a baldas virtuales. Una vez que una balda haya sido ordenada y clasificada se pueden empaquetar virtualmente. Cuando tengamos varios paquetes, podemos ordenar el orden de estos, y finalmente exportar las fotografías a un nuevo directorio conservando el orden y clasificación que habíamos realizado.
